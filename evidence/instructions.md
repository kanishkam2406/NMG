# Walkthrough Evidence Instructions

Please record a **60 to 90 second screen-recording** demonstrating the fully working AgileBoard Kanban application.

## What to Show in the Video
1.  **Demo Board Seeding**: Load the app, show the empty state, click **"Seed Demo Board"**, and show the pre-populated cards.
2.  **Card Manipulation**: Drag-and-drop a card from `To Do` to `In Progress` or `Done` to demonstrate fluid swimlane movement.
3.  **Card Customization**: Open a card modal, edit its title/description, change the assignee, toggle tags, set a due date, save it, and show the updated card.
4.  **Date Flagging**: Set a card's due date to a past date/time and show the red boundary highlights denoting it as overdue.
5.  **Board Management**: Click **"Create Board"**, name it, and show the newly created board with standard empty swimlanes.

## Saving the File
- Save the file as **`walkthrough.mp4`** (or a GIF).
- Commit and push it inside this directory: `/evidence/walkthrough.mp4`.
- **Note**: Do not use external Youtube/Loom links for submission as all evidence must live within the public repository.
